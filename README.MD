# La Guajira Salinity prediction

## Brief description
This repo is based in the Colombian open data about pluviosity and La Guajira Salinity. Hoping this can help to our proud La Guajira comunities to plan its watter needs.

Data should be found in:

<ul>
  <li><a href="https://www.datos.gov.co/dataset/Salinidad-Guajira/fwz7-px8j/about_data">Salinity</a></li>
  <li><a href="https://www.datos.gov.co/Ambiente-y-Desarrollo-Sostenible/Precipitaci-n/s54a-sgyg/about_data">Pluviosity</a></li>
</ul>

## Setup
To this repo we use python 3.10 an the libraries in //quirements.txt.
Using anaconda the setup is as follows:

`conda create -n guajira python==3.10`
`conda activate guajira`
`pip install -r requirements.txt`
